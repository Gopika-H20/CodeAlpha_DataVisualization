# Student Performance Analysis

## Project Overview
This project analyzes student performance using data analytics and visualization techniques. The goal is to identify factors affecting academic results.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Study time positively impacts student performance.
- Higher absences lead to lower grades.
- Strong correlation exists between G1, G2, and G3.
- Student behavior influences academic outcomes.

## Conclusion
This analysis helps understand key factors influencing student success and supports better academic decision-making.
